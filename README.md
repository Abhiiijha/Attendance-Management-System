# Student Attendance Management System (SQL)

## Overview

This project is a database-driven Student Attendance Management System implemented using MySQL. It focuses on designing relational schemas and executing structured queries to manage and analyze student attendance data efficiently.

## Objectives

* To design a normalized relational database schema
* To implement relationships using primary and foreign keys
* To perform analytical queries on attendance data
* To demonstrate practical understanding of SQL operations

## Database Structure

The system consists of the following tables:

### 1. Students

* Stores student details such as name, branch, year, gender, and hostel status

### 2. Faculty

* Contains information about faculty members and their departments

### 3. Subjects

* Maintains subject details and maps them with faculty and branch

### 4. Attendance

* Records attendance percentage of students for each subject

## Key Features

* Relational database design with proper constraints
* Use of primary and foreign keys
* Sample data insertion for testing
* Complex SQL queries for analysis and reporting

## Queries Implemented

The project includes multiple SQL queries such as:

* Students with attendance above a certain percentage
* Faculty teaching specific subjects
* Attendance shortage analysis
* Multi-condition filtering (branch, year, gender, etc.)
* Aggregation and grouping queries

## Technologies Used

* MySQL
* SQL (Structured Query Language)

## How to Run

1. Open MySQL Workbench or any MySQL client
2. Create a new database (if not already created)
3. Execute the SQL script file
4. Run the queries to view results


## Conclusion

This project demonstrates the implementation of a structured database system for managing student attendance, along with analytical queries to extract meaningful insights from the data.
